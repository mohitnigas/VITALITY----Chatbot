# VITALITY 🤖

**VITALITY** is an AI-powered desktop automation assistant that can read chat conversations, understand context, and generate intelligent, human-like responses using OpenAI's language models. Built with Python, it combines GUI automation, clipboard handling, and AI to streamline conversational workflows.

## ✨ Features

* 🧠 AI-generated contextual replies
* 💬 Understands ongoing chat history before responding
* ⚡ Automated message copying, processing, and sending
* 🖱️ Desktop automation using PyAutoGUI
* 📋 Clipboard integration with Pyperclip
* 🤖 Powered by OpenAI GPT models
* 🔄 Continuous monitoring for new messages
* 🐍 Simple and lightweight Python implementation

## 🛠️ Technologies Used

* Python
* OpenAI API
* PyAutoGUI
* Pyperclip
* Time Module

## 📌 How It Works

1. Monitors the chat window for new messages.
2. Copies the latest conversation from the screen.
3. Determines whether a reply is needed.
4. Sends the conversation to an OpenAI model for context-aware response generation.
5. Automatically pastes and sends the generated reply.
